# Welcome to my new blog!

This is the very first post of my Next.js 14 markdown blog. It is built using standard Markdown syntax and supports various formatting options.

## Features of this blog

Here is a list of features we will build:
* **Static Site Generation (SSG)**: Fast load times using Next.js static pages.
* **Tailwind CSS Styling**: Modern look with responsiveness and dark mode support.
* **Interactive Admin Panel**: Manage posts directly without accessing the codebase.
* **Client-side Search**: Instantly find articles using `fuse.js`.
